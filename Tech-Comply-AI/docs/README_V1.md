# Tech-Comply-AI: 法務・コンプライアンス特化型 LLM

ELYZA-japanese-Llama-2-7b をベースに、日本の法律（景品表示法、個人情報保護法、著作権法）に特化してファインチューニングを施した AI アシスタントです。広告のコンプライアンスチェックや利用規約のドラフト作成を支援します。

---

## プロジェクトの概要

本プロジェクトは、汎用的な LLM では対応が難しい「日本の法的解釈」と「厳格な JSON 出力」を両立させることを目的としています。実務レベルでの運用を想定し、出力形式の安定性と法的根拠の正確性を重視して開発されました。

### 主な機能
- **広告リスク判定**: 景品表示法に基づき、有利誤認や誇大広告のリスクを診断。
- **プライバシーポリシー・規約支援**: 個人情報保護法や著作権法に準拠した条文案の提示。
- **構造化データ出力**: システム連携を想定し、回答を厳格な JSON 形式で生成。

---

## 技術的な挑戦と改善 (v1 vs v2)

初期プロトタイプ（v1）での課題（出力の途切れ、JSON形式崩れ）を分析し、論理的な根拠に基づいてパラメータを最適化しました。

**主な改善ポイント要約:**

| 項目 | v1 (課題) | v2 (解決策) | 成果 |
| :--- | :--- | :--- | :--- |
| **データ量** | 35件 (不足) | **150件** | 汎用性の向上 |
| **文脈長** | 512 (途切れ) | **1024** | 回答の完結 |
| **学習深度** | LoRA Rank 16 | **LoRA Rank 32** | 複雑な論理の習得 |

## セットアップと使用方法

### 必要な環境

* Python 3.10+
* CUDA 対応 GPU (学習: A100推奨 / 推論: T4可)

### 1. インストール

pip install -r requirements.txt

### 2. 学習の実行

v2 設定ファイルを使用してファインチューニングを開始します。

python train_finetuning_v2.py --config_path ./config_v2.yaml

### 3. 推論テスト

python inference_v2.py --mode demo

## モデル使用

* ベースモデル: elyza/ELYZA-japanese-Llama-2-7b
* 学習手法: QLoRA (4-bit 量子化)
* ライブラリ: PEFT 0.9.0, Transformers
