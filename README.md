# genai-experiments

このレポジトリは、GPT-2を用いたチャットボット、Whisperによる音声認識、Diffusionモデルによる画像生成など、生成AIに関する実験をまとめています。

## 実験一覧
- GPT-2 Chatbot: src/gpt2_chatbot
- Whisper 音声認識: src/whisper_test
- Diffusion 画像生成: src/diffusion_test
