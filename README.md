# AI Sandbox (LLM / 生成AIの学習結果の倉庫)

このリポジトリは、大規模言語モデル（LLM）や生成AIの挙動を理解し、APIを利用した開発手法やプロンプトエンジニアリングを実践的に学習するための実験用コードを格納する場所です。

完成された商用アプリケーションではなく、技術検証、AIの出力の揺らぎの観察、およびPythonや最新のフロントエンド技術を用いた開発基盤の習得を主目的としています。

## 学習・検証の目的

本リポジトリ内の各プロジェクトを通じて、以下の技術要素の検証と習得を目指しています。

- **プロンプトエンジニアリングの実践:** Few-shotプロンプティングなどを活用し、AIの出力フォーマットやニュアンスを意図通りに制御する手法の探求。
- **モデルのファインチューニング:** 独自データセットを用いた学習による、LLMの性格調整や特定ドメイン（法令等）への適応検証。
- **ハイブリッド・アーキテクチャの構築:** ローカルAI（音声認識等）とクラウドAPI（LLM）を組み合わせ、機密性と処理速度を両立するセキュアなシステム設計の検証。
- **API連携と環境構築:** Google Gemini APIなどの外部AIサービスや、ローカルでの推論環境（Unsloth/PyTorch/faster-whisper等）を構築・処理する技術の習得。
- **モダンなUIとの統合:** React(Vite)やStreamlitなどのフレームワークを用い、検証結果を視覚的に確認・操作しやすいテストUI（フロントエンド）とバックエンド（FastAPI等）を連携させるフルスタック開発スキルの向上。

## 収録プロジェクト一覧

現在、以下の検証用アプリケーションおよびスクリプトが格納されています。各プロジェクトの詳細や検証結果については、それぞれのディレクトリ内のREADMEを参照してください。

- **[VoxNote](./VoxNote/)**
  - **概要:** ローカル環境での高精度な音声認識とクラウドLLMを組み合わせた、機密性重視のハイブリッド型AI議事録SaaS。
  - **検証テーマ:** 音声データや生体情報（声紋等）を外部に出さずにローカル（faster-whisper）でテキスト化し、テキストのみをクラウド（Gemini 2.5 Flash）で要約するセキュアなアーキテクチャの構築検証。および、ReactとFastAPIを連携させた非同期通信UIの実装。

- **[gen-mail](./gen-mail/)**
  - **概要:** Gemini APIを活用したビジネスメールの自動添削・生成アプリケーション。
  - **検証テーマ:** 複雑な状況設定（感情のバランスなど）に対するAIの言語化能力の評価と、出力されるテキストフォーマット（改行や段落）の厳密な制御テスト。

- **[Tech-Comply-AI](./Tech-Comply-AI/)**
  - **概要:** Webサービス開発における法的リスクを早期検知・修正提案する、日本の法令特化型AIアシスタント。
  - **検証テーマ:** Llama 2ベースモデルに対する独自データセットを用いたファインチューニング（4-bit QLoRA）。曖昧な表現を排除する性格調整と、法令・条文の引用に基づく推論の安定化検証。
 
- **[Inglish](./Inglish/)**
  - **概要:** ユーザーの興味（テーマ）と英語レベル（初心者〜上級者）に合わせて最適なリーディング記事を動的生成し、即時検索可能な辞書ウィジェットを備えたインプット特化型の学習モジュール。将来的な「包括的英語学習プラットフォーム」の手足となるコア機能の一つとして開発。
  - **検証テーマ:** ローカルベクトルDB（ChromaDB）を用いたRAG（検索拡張生成）による、文脈を考慮したLLMのパーソナライズ生成能力の検証。および、将来的なマイクロサービス的統合を見据えた、ReactとFastAPIによる「関心の分離（フロントエンド・バックエンドの完全独立）」アーキテクチャの実装テスト。

## 使用技術スタック（共通）

各実験プロジェクトで主に使用している技術・言語・モデルは以下の通りです。

- **プログラミング言語:** Python 3.x, JavaScript (React)
- **主要ライブラリ・フレームワーク:** FastAPI, React, Vite, Axios, Streamlit, python-dotenv, PyTorch, Unsloth
- **AI / LLM:** Google Generative AI (Gemini API), ELYZA-japanese-Llama-2, faster-whisper
- **データフォーマット:** JSON, JSONL, Markdown, 音声ファイル (mp3等)

## 運用ポリシー

本リポジトリは個人の技術検証および学習プロセスを記録するサンドボックス環境です。
格納されているコードはテスト用途であり、本番環境での稼働やセキュリティの完全性を担保するものではありません。APIキーなどの機密情報を含む設定ファイル（`.env` 等）はバージョン管理に含めず、ローカル環境のみで厳格に管理しています。
